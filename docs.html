Set the payment systems

Set the payment systems of the promotion. If the payment systems list is empty, the promotion will be applied for all payment systems. If the promotion is read-only (read_only = true), you are not allowed to call this command.

HTTP REQUEST

PUT https://api.xsolla.com/merchant/merchants/{mer- chant_id}/promotions/{promotion_id}/payment_systems

merchant_id
int
ID merchant.
TypePATCH
Modifies only specified fields of the existing entity that matches the identifier you provide.
user.country.value
string
User country. 2-letter definition (in uppercase) of the country is used according to ISO 3166-1 alpha-2 standard.
